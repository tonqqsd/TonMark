# Ship calmer Markdown

TonMark is built for writers who keep real work in folders: drafts, notes, chapters, specs, and research files.

## Why it feels fast

- Open a folder once and move through Markdown files from the native sidebar.
- Jump with Quick Open instead of hunting through long trees.
- Search the whole workspace without leaving the editor.
- Keep local snapshots before large rewrites.
- Switch between Live, Source, and Reading modes based on the task.

## A small workflow

1. Draft in Live mode.
2. Use Source mode when Markdown structure matters.
3. Review in Reading mode.
4. Export HTML or PDF when the piece is ready.

> Private by default. Local files stay local unless you decide otherwise.
